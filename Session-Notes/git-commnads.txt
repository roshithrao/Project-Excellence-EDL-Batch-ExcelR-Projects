Signup on Github and login with your credentials

Click the + sign at the top right corner to create a new repository. Repositories are like your code folders online.

Name your repository and give it a description (this is optional) and give it public access.

Click the "Create repository" button to create the repository. 

git config --global user.name "your_username"

git config --global user.email "your email"

git init

git add --all

git commit -m "Your commit message"

git remote add origin https://github.com/tutorials/hello-world.git

git push origin -u master
